# Mineciska
Утилитарный клиент специально для сервера [MINECRAFT СНГ](https://minecis.net/). Нажмите `Правый шифт` для того, чтобы открыть меню в игре.

## О моде
Mineciska форк мода [Asteroid](https://github.com/optipack/asteroid), клиентский мод на основе Fabric.

Форк был сделан для исключения тотема из Auto Replenish т.к. это запрещено правилами сервера.

### Заслуги
Изначальная разработка мода лежит на команде Asteroid (optipack).

Обязательно поставьте им звезду на [Github](https://github.com/optipack/asteroid).

Updated waypoint textures by Wulfian.

## Функции
### Модули
Ниже приведен список модулей, включенных в мод. Большинство из них имеют ту же функциональность, что и другие моды на сайте, поэтому я решил, что их можно оставить.

Модули, отмеченные звездочкой (`*`), требуют права OP или одиночную игру для использования.

#### Игрок
- **Auto Clicker**: Автоматические клики по таймеру. Полезно для фарма мобов.
- **Auto Replenish**: Автоматически заменяет инструмент / еду на хотбаре из инвентаря при исчерпании. (Исключение: Тотем) Удобное улучшение QoL.
- **Auto Respawn**: Автоматически нажимает кнопку возрождения после смерти.
- **Auto Tool**: Автоматически переключает на нужный инструмент при добыче блока.
- **Fake Player**: Создает полностью клиентского фейкового игрока. Практического применения почти нет, кроме красивых скриншотов.
- **Middle Click Extra**: Назначает средней кнопке мыши использование фейерверка, бутылочки опыта или добавление друга.
- **Name Protect**: Скрывает ваш никнейм и скин. Полезно для скриншотов без раскрытия аккаунта.
- **No Interact**: Позволяет запретить взаимодействие / ломание определенных блоков. Полезно при добыче аметиста.
- **No Mining Trace**: Позволяет ломать блоки, даже если их перекрывает сущность.
- **Portals**: Позволяет открывать инвентарь, находясь в портале Нижнего мира.
- **Rotation**: Позволяет зафиксировать yaw и/или pitch игрока.

#### Рендер
- **Better Tab**: Улучшенное и настраиваемое меню TAB.
- **Better Tooltips**: Предпросмотр шалкеров, эндер-сундуков и прочего.
- **Block ESP***: Подсвечивает выбранные блоки из белого списка.
- **Block Selection**: Изменяет отображение выделения блоков.
- **Blur**: Размывает фон некоторых меню.
- **Breadcrumbs**: Оставляет линию позади игрока для отслеживания пути.
- **Break Indicators**: Улучшает индикаторы разрушения блоков.
- **Camera Tweaks***: Настройка камеры от третьего лица, включая прохождение камеры сквозь блоки.
- **Entity Owner**: Показывает владельца прирученного существа.
- **Free Look**: Позволяет свободно двигать камерой без изменения направления игрока.
- **Freecam***: Освобождает камеру, как в режиме наблюдателя.
- **Fullbright**: Полностью отключает освещение.
- **Hand View**: Настройка отображения предметов в руке.
- **Item Highlight**: Подсвечивает определенные предметы в интерфейсах.
- **Item Physics**: Предметы лежат на земле реалистично.
- **Light Overlay**: Показывает уровни освещения на блоках.
- **Marker**: Создает клиентские голограммы для планирования построек.
- **Nametags***: Настраиваемые никнеймы над сущностями.
- **No Render**: Отключает отображение тумана, частиц, сущностей и прочего.
- **Tracers***: Линии от прицела к сущностям.
- **Trail**: Частицы за игроком.
- **Unfocused CPU**: Ограничивает FPS, когда окно Minecraft неактивно.
- **Waypoints**: Создание и сохранение точек маршрута.
- **Xray***: Отключает рендер определенных блоков.
- **Zoom**: Увеличение обзора.

#### Разное
- **Ambience**: Настройка визуального окружения.
- **Auto Reconnect**: Автоматическое переподключение к серверу.
- **Auto Sign**: Автоматически копирует текст на таблички.
- **Better Chat**: Улучшения чата, фильтры, префиксы.
- **Discord Presence**: Настраиваемый Discord Rich Presence.
- **Inventory Tweaks**: Автосортировка, перенос предметов и улучшения инвентаря.
- **Sound Blocker**: Точная настройка отключения определенных звуков.

### Команды
Следующие команды Mineciska используют префикс `.`:

- **Dismount (`.dismount`)**: Покинуть транспорт.
- **Disconnect (`.disconnect`)**: Отключиться от сервера.
- **Drop (`.drop`)**: Выбросить предметы.
- **Enchant (`.enchant`)**: Зачаровать предмет (только Creative).
- **Fake Player (`.fake-player`)**: Управление фейковыми игроками.
- **Friend (`.friend`)**: Управление друзьями.
- **Help (`.help`)**: Список команд.
- **NBT (`.nbt`)**: Изменение NBT предмета (только Creative).
- **Peek (`.peek`)**: Просмотр содержимого контейнера.
- **Ender Chest (`.ec`)**: Просмотр эндер-сундука.
- **Profiles (`.profiles`)**: Управление профилями.
- **Reload (`.reload`)**: Перезагрузка Mineciska.
- **Reset (`.reset`)**: Сброс настроек.
- **Say (`.say`)**: Отправка сообщения с поддержкой StarScript.
- **Server (`.server`)**: Информация о сервере.
- **Toggle (`.toggle`)**: Переключение модулей.
- **Settings (`.settings`)**: Настройки модулей.
- **Save Map (`.sm`)**: Сохранение карты как изображения.
- **Macro (`.macro`)**: Запуск макроса.
- **Modules (`.modules`)**: Список модулей.
- **Binds (`.binds`)**: Список биндов.
- **Give (`.give`)**: Выдать предмет (только Creative).
- **Bind (`.bind`)**: Назначить клавишу модулю.
- **FOV (`.fov`)**: Настройка угла обзора.
- **Rotation (`.rotation`)**: Настройка вращения.
- **Waypoint (`.wp`)**: Управление точками маршрута.
- **Input (`.input`)**: Симуляция ввода.

#### Макросы
Mineciska позволяет создавать макросы, объединяющие команды в последовательности.
Их можно использовать для автоматизации задач и привязывать к горячим клавишам.
